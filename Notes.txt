Pydantic Models only knows how to deal with the dictionary and SQL.

Schema/Pydantic Models is to ensure that it is requesting what is provided by the user and in return response is given.

SQLalchemy models is for handling and forming a database table like CRUD application table in database.

In schema, we can also define a class that just need a specific fields for the action/response.

models = this file has the table format for the database.

database = this file contains information and connects the local databse with API.
